# 🚀 AI Automation Platform

A premium, responsive AI-powered SaaS landing page developed for the **Frontend Battle Hackathon**. This project showcases modern frontend development practices with a focus on responsive design, dynamic pricing, SEO optimization, accessibility, and smooth user interactions.

---

## ✨ Features

* 🎨 Modern & Responsive AI SaaS Landing Page
* 💰 Dynamic Pricing (Monthly & Annual Billing)
* 🌍 Multi-Currency Support (INR, USD, EUR)
* 🧩 Bento Grid Layout (Desktop)
* 📂 Accordion Layout (Mobile)
* 🔄 State Persistence on Screen Resize
* ⚡ Smooth CSS Animations & Transitions
* 🔍 SEO Optimized
* ♿ Semantic HTML & Accessibility
* 🚀 Performance Optimized

---

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* HTML5
* CSS3

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sayaliTechX/AI-Automation-Platform-Frontend-Battle.git
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
